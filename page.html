The first step in understanding how to position elements is to understand the fundamental model of CSS - The box model

The Box Model

Every element you place on a page creates a box. You cannot always see the boundaries and the borders of the boxes you create, but the boxes are there. A div element creates a box, as do h1, img, span, and td elements. Each box has properties you can set with CSS. I've illustrated some these properties in Figure 1.

Figure 1 Border, padding, array

The padding of a box is the distance between the content of the box and the box's border, and the margin is the distance between a box and any adjacent element. In other words, padding adds space inside the border, while a margin adds space outside the border. In the following CSS, I set both the padding and the margin to a distance of 10 pixels.
